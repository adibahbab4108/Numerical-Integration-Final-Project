Adib's- Numerical Integration Calculator Documentation

Introduction

This project is designed to perform numerical integration using three different methods: the Trapezoidal Rule, Simpson's 1/3 Rule, and the Simpson's 1/3 Rule. The calculator takes integration bounds as input, and then applies each of these rules to approximate the integral. The results from the three methods are displayed side by side, allowing users to compare the accuracy and efficiency of each approach. This comparison not only highlights the strengths and weaknesses of the different numerical techniques but also provides valuable insights into their practical applications. Whether for educational purposes or practical problem-solving, this Numerical Integration Calculator serves as a versatile and user-friendly resource for anyone working with integral approximations.

Features

1. Input Upper limit, lower limit and interval
2. Analyze Button
3. Reset Button
4. Result Comparison with PieChart and BarChart

Implemented Methods

1. Trapezoidal Rule
2. Simpon’s 1/3 (One-third) Rule
3. Simpson’s 3/8 (Three-eight) Rule

Graphical Analysis

1. Pie Chart - Iteration Convergence Shows the percentage of iterations needed for convergence.
2. Bar Chart - Time Taken Displays the time taken for each method to solve the system.

General Quadrature formula for equidistant ordinates

Suppose y = f(x) is a function whose values are known at equidistant values of x i.e. f(x 0 ) = y 0 , f(x0 +h) = y 1 , f(x0 +2h) = y 2 , ……., f(x 0 +nh) = yn. To evaluate I, we have to replace f(x) by a suitable interpolation formula. Let the interval [a, b] be divided into n subintervals with the division points a = x0 < x0+h < …… < x0 +nh = b where h is the width of each subinterval.

Approximating f(x) by Newton’s forward interpolation formula, we get

                
                  I = h[ny0 + (n2/2)Δy0 + (n3/3-n2/2)Δ2y0/2! + (n4/4-n3+n2)Δ3y0/3! +…]
                
                
                  where h = (b-a)/n, y0 = f(a), yn = f(b)

Trapezoidal Rule

Substituting n =1 in the general quadrature formula and neglecting all differences greater than the first, we get 

        I = h/2[y0 + 2(y1+ y2 + ………. + yn-1) + yn]

Let's consider the example:

Calculate the value of ∫01 x / (1 + x) dx correct up to 3 significant figures taking six intervals by trapezoidal rule.

Solution: Here we have, f(x) = x / (1 + x), a = 0, b = 1, n = 6

∴ h = (b - a) / n = (1 - 0) / 6 = 1 / 6

Simpson's 1/3 Rule

Substituting n = 2 in the general quadrature formula and neglecting the third and other higher order differences, we get 

        
          I = h/3[y0 + 4(y1 + y3 + y5 + ……. + yn-1) + 2(y2 + y4 + ……. + yn-2) + yn]

Similarly, if we calculate for the above example by Simpson's 1/3 rule, We get the result I = 0.306830, which is very close to the result of Traphizoidal rule.

Simpson 3/8 Rule

Substituting n = 3 in the general quadrature formula and neglecting all the differences above 

          I = 3h/8 [(y0 + yn) + 3(y1 + y2 + y4 + y5 + … + yn-2 + yn-1) + 2(y3 + y6 + … + yn-3)]

And if we calculate the above example with Simpson's 3/8 rule, we get the value I=0.306805

which is again very close to the result of previous two rules!
